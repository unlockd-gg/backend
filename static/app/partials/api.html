API DOCUMENTATION

There are two aspects to the UETOPIA API. They are used for different purposes, and require different authentication.

User authenticated endpoints

These are used for things that users can do. You need a valid authentication token in order to use these. Endpoints marked "PROTECTED" can only be used from uetopia.com. All other endpoints may be used on a third-party website.

The Endpoints portal is currently down.

Server authenticated endpoints

These are used for things that only dedicated servers are allowed to do. Authentication is the server API key/secret.

/api/v1/server/info

Notify the backend that the server is ready. Retrieve server specific details.
